# Learning Hub Skill

## Overview
Track learning progress, manage courses, build knowledge base, and achieve skill development goals.

## When to Use
- User wants to learn something new
- User asks about courses/tutorials
- User wants study plan
- User mentions belajar/skill baru

## Capabilities

### 1. Course Tracking
- Track enrolled courses
- Progress percentage
- Completion certificates
- Learning streaks

### 2. Skill Mapping
- Current skills inventory
- Skill gaps analysis
- Learning paths
- Competency levels

### 3. Knowledge Base
- Save key learnings
- Create summaries
- Link related concepts
- Spaced repetition reminders

### 4. Learning Goals
- Set learning targets
- Track milestones
- Time allocation
- Priority based on goals

## Commands

### Courses
- add course [name] from [platform]
- course progress
- complete module [name]
- learning stats

### Skills
- my skills / skill aku
- skill gaps for [role/goal]
- learning path for [skill]
- skill level [skill]

### Knowledge
- summarize what I learned about [topic]
- review [topic]
- connect [topic A] with [topic B]

### Goals
- learning goal: [goal]
- study plan for [topic]
- time to learn [skill]
