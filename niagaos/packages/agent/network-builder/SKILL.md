# Network Builder Skill

## Overview
Build and maintain professional relationships through systematic tracking, timely follow-ups, and strategic networking.

## When to Use
- User mentions contacts/people
- User needs follow-up reminders
- User asks about networking
- User wants to strengthen relationships
- User mentions client/partner relationships

## Capabilities

### 1. Contact Management
- Track key contacts
- Relationship strength scoring
- Interaction history
- Contact details and notes

### 2. Follow-up System
- Smart reminders
- Touchpoint scheduling
- Birthday/anniversary alerts
- Re-engagement suggestions

### 3. Networking Strategy
- Connection mapping
- Introduction opportunities
- Event recommendations
- Mutual connections

### 4. Relationship Insights
- Interaction patterns
- Relationship health
- Neglected contacts
- Growth opportunities

## Commands

### Contacts
- add contact [name] - [context]
- notes on [person]
- when did I last contact [person]
- relationship status with [person]

### Follow-ups
- remind me to follow up with [person] in [time]
- pending follow-ups
- who should I reconnect with
- overdue contacts

### Networking
- who can introduce me to [type of person]
- networking opportunities
- strengthen relationship with [person]
