# Deep Work Skill

## Overview
Maximize productivity with structured focus sessions, distraction management, and work pattern analysis.

## When to Use
- User wants to focus/concentrate
- User mentions pomodoro/focus timer
- User feels distracted
- User wants productivity tips
- User asks nak start kerja

## Capabilities

### 1. Focus Sessions
- Start timed focus blocks
- Pomodoro technique (25/5 or custom)
- Deep work blocks (90-120 min)
- Break reminders

### 2. Distraction Management
- DND mode activation
- Notification batching
- Social media blocking tips
- Environment suggestions

### 3. Progress Tracking
- Time logged per task
- Focus score
- Daily/weekly patterns
- Streak tracking

### 4. Accountability
- Check-ins during sessions
- Session summaries
- Goal tracking
- Reflection prompts

## Commands

### Focus
- start focus [duration]
- pomodoro [task]
- deep work on [project]
- end session

### Tracking
- how long on [task]
- productivity today
- focus streak
- weekly review

### Manage
- do not disturb
- batch notifications
- schedule focus blocks

## Session Types

### Pomodoro (25+5)
- 25 min work
- 5 min break
- After 4 cycles: 15-30 min long break

### Deep Work (90-120 min)
- Extended focus block
- No interruptions
- Complex/creative tasks

### Sprint (50+10)
- 50 min work
- 10 min break
- Good for meetings-heavy days
