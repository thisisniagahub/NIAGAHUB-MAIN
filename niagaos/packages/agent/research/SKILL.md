# Research Skill

## When to Use
- User asks for research on a topic
- User needs comprehensive information
- User wants comparison/analysis

## Approach
1. Break down the research question
2. Search multiple sources (if web search enabled)
3. Synthesize findings
4. Present with citations

## Output Format
### [Topic]
**Summary:** Brief overview

**Key Findings:**
- Point 1
- Point 2

**Sources:**
- [Source 1](url)
- [Source 2](url)

**Recommendations:**
Based on research...

## Commands
- Research [topic]: Full research report
- Quick facts [topic]: Brief summary
- Compare [A] vs [B]: Comparison analysis
