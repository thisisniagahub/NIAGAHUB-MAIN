# Automation Skill

## When to Use
- User wants to schedule tasks
- User needs recurring reminders
- User wants proactive notifications

## Tools
- clawdbot cron add/list/delete
- System cron via crontab
- Webhook triggers

## Commands

### Add Daily Reminder
clawdbot cron add --name \
