# Business Intelligence Skill

## Overview
Provide actionable business insights, market research, competitor monitoring, and strategic analysis for NiagaHub and other ventures.

## When to Use
- User asks about market trends
- User wants competitor analysis
- User needs business strategy advice
- User asks apa orang lain buat
- User needs pricing/positioning research

## Capabilities

### 1. Market Research
- Industry trends via web search
- Market size estimation
- Growth projections
- Regional analysis (focus: SEA, Malaysia)

### 2. Competitor Analysis
- Feature comparison
- Pricing analysis
- Positioning maps
- SWOT analysis
- Social media presence

### 3. Trend Tracking
Track and summarize:
- Tech trends (AI, SaaS, etc)
- Industry news
- Startup funding rounds
- Product launches

### 4. Strategic Insights
- Opportunity identification
- Threat assessment
- Go-to-market suggestions
- Pricing strategy

## Commands

### Research
- research [topic/company/industry]
- competitor analysis [company]
- market size for [product/service]
- trends in [industry]

### Tracking
- track [company/keyword]
- news about [topic]
- funding rounds in [sector]

### Strategy
- SWOT for [business/product]
- positioning for [product]
- pricing strategy for [service]

## Data Sources
1. Web search (Brave API)
2. Public financial data
3. Industry reports
4. Social media signals
5. App store rankings
